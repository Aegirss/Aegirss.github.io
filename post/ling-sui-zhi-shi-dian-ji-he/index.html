可删除的线性基（强制离线）

考虑维护一个支持插入、删除元素的线性基，值域为[0,2n−1][0,2^n-1][0,2n−1]。

因为强制离线，我们先预处理出每个元素的删除时间，并在线性基里维护这个时间。

每次加入一个元素，除了正常的线性基插入过程，还需要替换掉删除时间较早的。

删除元素的时候只需要找到删除时间与当前时间相等的值，然后正常删除即可。

参考代码：

//v表示插入的元素，t表示这个元素的删除时间
void ins(int v, int t)
{
	for(int i = n - 1; ~i; --i)
		if((v >> i) & 1)
		{
			if(!b[i]) {b[i] = v, tim[i] = t; ++siz; return;} //siz表示线性基大小
			if(t > tim[i]) {swap(b[i], v); swap(tim[i], t);} //替换掉删除时间早的
			v ^= b[i];
		}
}

//t表示当前的时间
void del(int t)
{
	for(int i = n - 1; ~i; --i)
		if(tim[i] == t) //找到删除时间与当前时间相等的值
		{
			tim[i] = b[i] = 0;
			--siz;
			return;
		}
}
