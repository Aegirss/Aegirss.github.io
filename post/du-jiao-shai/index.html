杜教筛

2020-02-08 8 min read # 数论

杜教筛是一种在非线性时间内求积性函数前缀和的算法。

前置知识：

积性函数：

数论函数：定义域为正整数的函数。

积性函数：对于任意互质的整数a,ba,ba,b，有f(ab)=f(a)f(b)f(ab) = f(a)f(b)f(ab)=f(a)f(b)的数论函数

完全积性函数：对于任意整数a,ba,ba,b，有f(ab)=f(a)f(b)f(ab) = f(a)f(b)f(ab)=f(a)f(b)的数论函数

常见的积性函数：ϕ,μ,σ,d\phi, \mu, \sigma, dϕ,μ,σ,d（分别表示：欧拉函数，莫比乌斯函数，因子和函数，因子个数函数）

常见的完全积性函数：ϵ,I,id\epsilon, I, idϵ,I,id（ϵ(n)=[n=1],I(n)=1,id(n)=n\epsilon(n) = [n = 1], I(n) = 1, id(n) = nϵ(n)=[n=1],I(n)=1,id(n)=n）

狄利克雷卷积：
设f,gf,gf,g是两个数论函数，它们的狄利克雷卷积为：(f∗g)(n)=∑d∣nf(d)g(nd)(f*g)(n) = \sum_{d|n}f(d)g(\frac{n}{d})(f∗g)(n)=∑d∣n​f(d)g(dn​)
运算律：满足交换律，结合律
单位元：ϵ\epsilonϵ（f∗ϵ=ff * \epsilon = ff∗ϵ=f）

几个重要的性质：
μ∗I=ϵ\mu * I = \epsilonμ∗I=ϵ
ϕ∗I=id\phi * I = idϕ∗I=id
μ∗id=ϕ\mu * id = \phiμ∗id=ϕ

g(n)=∑d∣nf(d)=>g=f∗Ig(n) = \sum_{d|n}f(d) => g = f * I g(n)=d∣n∑​f(d)=>g=f∗I

f(n)=∑d∣nμ(d)g(nd)=>μ∗g=μ∗I∗f=ϵ∗f=ff(n) = \sum_{d|n}\mu(d)g(\frac{n}{d}) => \mu * g = \mu * I * f = \epsilon * f = f f(n)=d∣n∑​μ(d)g(dn​)=>μ∗g=μ∗I∗f=ϵ∗f=f

杜教筛：

下面就是正题了，我们现在要求积性函数f(i)f(i)f(i)的前缀和，设∑i=1nf(i)=S(n)\sum_{i=1}^{n} f(i) = S(n)∑i=1n​f(i)=S(n)
再找一个积性函数ggg，考虑g∗fg * fg∗f的前缀和：

∑i=1n(g∗f)(i)\sum_{i=1}^{n} (g * f)(i) i=1∑n​(g∗f)(i)

=∑i=1n∑d∣ig(d)∗f(id)= \sum_{i=1}^{n} \sum_{d|i} g(d) * f(\frac{i}{d}) =i=1∑n​d∣i∑​g(d)∗f(di​)

=∑d=1ng(d)∑i=1ndf(did)= \sum_{d=1}^{n} g(d) \sum_{i=1}^{\frac{n}{d}} f(\frac{di}{d}) =d=1∑n​g(d)i=1∑dn​​f(ddi​)

=∑d=1ng(d)S(nd)= \sum_{d=1}^{n} g(d) S(\frac{n}{d}) =d=1∑n​g(d)S(dn​)

上面这个式子启发我们去考虑g(1)S(n)g(1)S(n)g(1)S(n)，容易得到：

g(1)S(n)=∑i=1ng(i)S(ni)−∑j=2ng(j)S(nj)g(1)S(n) = \sum_{i=1}^{n} g(i) S(\frac{n}{i}) - \sum_{j=2}^{n} g(j) S(\frac{n}{j}) g(1)S(n)=i=1∑n​g(i)S(in​)−j=2∑n​g(j)S(jn​)

即：

g(1)S(n)=∑i=1n(g∗f)(i)−∑j=2ng(j)S(nj)g(1)S(n) = \sum_{i=1}^{n} (g * f)(i) - \sum_{j=2}^{n} g(j) S(\frac{n}{j}) g(1)S(n)=i=1∑n​(g∗f)(i)−j=2∑n​g(j)S(jn​)

至此，我们已经得到了杜教筛的核心式子。

假如我们选取一个合适的积性函数ggg，使得我们能够快速求解ggg的前缀和以及g∗fg * fg∗f的前缀和，那么我们就可以利用数论分块+递归求解上面的式子。

ll GetSum(int n) { // 算 f 前缀和的函数
  ll ans = f_g_sum(n); // 算 f * g 的前缀和
  // 以下这个 for 循环是数论分块
  for(ll l = 2, r; l <= n; l = r + 1) { // 注意从 2 开始
    r = (n / (n / l)); 
    ans -= (g_sum(r) - g_sum(l - 1)) * GetSum(n / l);
    // g_sum 是 g 的前缀和
    // 递归 GetSum 求解
  } return ans; 
}

可以证明，这样做的复杂度是O(n34)O(n^{\frac{3}{4}})O(n43​)的。
我们还可以进一步优化，即先线性筛出前mmm个答案，然后再利用杜教筛求解。
当m=n23m = n^{\frac{2}{3}}m=n32​时，复杂度是O(n23)O(n^{\frac{2}{3}})O(n32​)

模板题：杜教筛

我们先考虑μ\muμ的前缀和，注意到μ∗I=ϵ\mu * I = \epsilonμ∗I=ϵ，于是我们取g=I,f=μg = I, f = \mug=I,f=μ。
那ϵ,I\epsilon, Iϵ,I的前缀和都是可以O(1)O(1)O(1) 求的，直接套用杜教筛的式子即可。
然后是求ϕ\phiϕ的前缀和，因为有ϕ∗I=id\phi * I = idϕ∗I=id，所以取g=I,f=ϕg = I, f = \phig=I,f=ϕ。
这里要注意一下，当n=231−1n = 2^{31} - 1n=231−1的时候可能会在计算过程中爆llllll，转换成ullullull运算即可。

#include<bits/stdc++.h>
#include <tr1/unordered_map>
using namespace std;
using namespace std::tr1;

typedef long long ll;
typedef unsigned long long ull;

const int inf = 0x7FFFFFFF;
const int MAXN = 7000000;

int cnt, prim[MAXN + 10];
bool v[MAXN + 10];
ll mu[MAXN], phi[MAXN];
unordered_map<int, ll> ansmu, ansphi;

void init()
{
    v[1] = mu[1] = phi[1] = 1;
    for(int i = 2; i <= MAXN; ++i)
    {
        if(!v[i]) prim[++cnt] = i, mu[i] = -1, phi[i] = i - 1;
        for(int j = 1; j <= cnt && i * prim[j] <= MAXN; ++j)
        {
            v[i * prim[j]] = 1;
            if(i % prim[j]) mu[i * prim[j]] = -mu[i], phi[i * prim[j]] = phi[i] * phi[prim[j]];
            else {mu[i * prim[j]] = 0, phi[i * prim[j]] = phi[i] * prim[j]; break;}
        }
    }
    for(int i = 2; i <= MAXN; ++i) mu[i] += mu[i - 1], phi[i] += phi[i - 1];
}

ll sum_mu(int n)
{
    if(n <= MAXN) return mu[n];
    if(ansmu[n]) return ansmu[n];
    ll ret = 0;
    for(int l = 2, r = 0; r < inf && l <= n; l = r + 1)
    {
        r = n / (n / l);
        ret += (r - l + 1) * sum_mu(n / l);
    }
    return ansmu[n] = 1ll - ret;
}

ll sum_phi(int n)
{
    if(n <= MAXN) return phi[n];
    if(ansphi[n]) return ansphi[n];
    ll ret = 0;
    for(int l = 2, r = 0; r < inf && l <= n; l = r + 1)
    {
        r = n / (n / l);
        ret += (r - l + 1) * sum_phi(n / l);
    }
    return ansphi[n] = (ull)n * (n + 1ull) / 2ull - ret;
}

int main()
{
    init(); int T, n;
    scanf("%d", &T);
    while(T--)
    {
        scanf("%d", &n);
        printf("%lld %lld\n", sum_phi(n), sum_mu(n));
    }
    return 0;
}
